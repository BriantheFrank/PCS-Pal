How to Use This Tool

Once signed in, work through the move in order so the details stay connected.

1. Start with the checklist

Build the core move plan and keep the family and service-member tasks visible.

2. Organize the move inventory

Capture rooms, boxes, and high-value items before pack-out so the inventory is usable for planning and claims.

3. Lock in logistics

Track movers, itinerary, and key handoff dates so the travel plan and delivery plan stay aligned.

Built for service members and military families

The landing page stays public. The planning workspace stays behind sign-in so PCS data can sync across devices without exposing it publicly.
